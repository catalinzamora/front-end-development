Proprietăţile width şi height se folosesc la definirea lăţimii şi înălţimii conţinutului elementelor, respectiv diferite elemente HTML deţin diferite valori implicite pentru înălţime şi lăţime. De exemplu, într-una din lecţiile anterioare am prezentat, ca o procedură implicită, că imaginile din cadrul documentului HTML au dimensiunea lor inițială. Pe de altă parte, paragrafele ocupă întregul spaţiu disponibil în lăţime. Despre astfel de caracteristici ale diferitor elemente HTML vom vorbi mai mult în următorul modul, iar acum vom dedica timp primului dintre elementele CSS Box Model, respectiv diferitor moduri de definire a înălţimii şi lăţimii.
